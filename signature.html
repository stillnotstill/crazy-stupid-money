How to Install in Gmail

  1. Open this file in Chrome
  2. Select the signature in the white box above (from the Franklin image through crazystupid.money)
  3. Copy (Cmd+C)
  4. Gmail → Settings → See all settings → Signature
  5. Create new or edit existing, paste (Cmd+V)
  6. Scroll down, Save Changes

The Franklin image is hosted on Netlify. Gmail caches it on first send.
